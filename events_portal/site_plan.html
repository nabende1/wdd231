Community Events Portal - Site Plan

Site Name

Community Events Hub

This name was chosen because it represents a central space where community members can view, share, and engage with local events, making it easy to discover activities nearby.

Site Purpose

The purpose of the Community Events Portal is to serve as a comprehensive directory of events happening in the local area. It provides event listings, submission options for new events, and contact details for event organizers.

Scenarios

  • How can I submit an event for the community to see?
  • Where can I find upcoming events in my area?

Color Schema

The chosen color scheme enhances accessibility and provides a welcoming, vibrant feel to the site.

  • Background Color: Light green #f3f9ed
  • Text and Headers: Dark charcoal #292c35
  • Accent Color: Blue for links and accents #2972fb
  • Buttons: Pink-red for interactive elements #fa316c

Typography

The primary font will be Verdana for body text for readability, while Arial will be used for headers to maintain a clean and structured layout.

  • Body Text: Verdana
  • Headers: Arial
